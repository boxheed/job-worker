# job-worker

A command-line executable Node.js package.

## Installation

You can install this package directly from GitHub:

```bash
npm install git+https://github.com/boxheed/job-worker.git
```

Or run it directly using `npx`:

```bash
npx git+https://github.com/boxheed/job-worker.git
```

## Usage

```bash
job-worker [name]
```

Example:
```bash
job-worker Jules
# Output: Hello, Jules!
```

## Development

### Prerequisites

- npm

### Setup

```bash
npm install
```
